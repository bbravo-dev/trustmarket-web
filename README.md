
# TrustMarket

## Overview
TrustMarket is a marketplace MVP focused on peer-to-peer transactions,
allowing users to create listings with images and manage their posts securely.

## Features
- Email-based authentication using Supabase Auth
- Create, edit and delete marketplace listings
- Image upload and storage with Supabase Storage
- Basic real-time chat for buyerâ€“seller communication
- Protected routes for authenticated users
- Responsive UI for desktop and mobile

## Tech Stack
- React
- TypeScript
- Supabase
- Tailwind CSS
